## About The Project

This project seeks to create a mathematical expression evaluator using C++. The program will have the 
functionalities of addition, subtraction, multiplication, division, modulus, and exponentiation. The program 
should be able to run on any input and tell the user what errors occur. The project should include a user-
friendly graphical interface and intuitive use.
